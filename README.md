Πλατφόρμα Σχολικών Εκδρομών (School-Trips Platform)
Μάθημα: Ψηφιακό Εκπαιδευτικό Υλικό

Αυτό το αποθετήριο περιέχει τον σχεδιασμό και την έρευνα για την πλατφόρμα School-Trips, μια καινοτόμα λύση που στοχεύει στην αναβάθμιση της εμπειρίας των σχολικών εκδρομών. Διαπιστώνοντας ότι οι σημερινές εκδρομές είναι συχνά μονότονες και δεν αξιοποιούνται στο μέγιστο, η εφαρμογή αυτή προσφέρει ένα ενοποιημένο σύστημα επικοινωνίας και οργάνωσης για όλους τους εμπλεκόμενους: μαθητές, γονείς, καθηγητές, σχολεία και συνεργαζόμενους φορείς.
Βασικά Στοιχεία του Project:

Έρευνα Χρηστών (User Research): Διεξαγωγή συνεντεύξεων με μαθητές όλων των βαθμίδων (Δημοτικό, Γυμνάσιο, Λύκειο) για την καταγραφή αναγκών, όπως η ζήτηση για πιο διαδραστικές εκδρομές με μεγαλύτερη σύνδεση στο γνωστικό αντικείμενο.
Σχεδιασμός UI/UX (Figma): Ανάπτυξη διαδραστικών πρωτοτύπων (prototypes) με γνώμονα την ευχρηστία, την εύκολη πλοήγηση των καθηγητών (π.χ. γρήγορη δημιουργία εκδρομής) και την άμεση ενημέρωση των γονέων.
Διαχείριση Ενδιαφερομένων (Stakeholders): Ειδικά σχεδιασμένες λειτουργίες για την ασφάλεια (γονείς), τον οικονομικό έλεγχο (σχολεία) και την ομαλή διεξαγωγή των δραστηριοτήτων (εταιρείες/φορείς).

School-Trips Platform 
Course: Digital Educational Content

This repository contains the design and user research for the School-Trips platform, an innovative solution aimed at upgrading the school trip experience. Recognizing that current field trips can be monotonous and underutilized, this application provides a unified communication and organization system for all stakeholders: students, parents, teachers, schools, and partner companies.

Key Project Highlights:

User Research: Conducted interviews with students across all educational levels (Primary, Middle, and High School) to identify key needs, such as the demand for more interactive trips closely tied to their curriculum.
UI/UX Design (Figma): Developed interactive prototypes focused on usability, enabling seamless navigation for teachers (e.g., quick trip creation) and real-time updates for parents.
Stakeholder Management: Tailored features ensuring safety (for parents), financial oversight and compliance (for schools), and smooth operational logistics (for companies/agencies).
